## Getting started

1. Clone the repo
   ```sh
   git clone https://github.com/vinhht2703/quod-ai-challenge.git
   ```
2. Restore packages
   ```
   npm install
   ```
3. Build and run demo
   ```
   npm start
   ```
4. Runs the app in the development mode.\
   Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
